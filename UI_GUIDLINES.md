# UI Guidelines

## Design Style

JagoCurhat visual style:

- Minimal
- Cute cartoon
- Threads-inspired
- Dark mode
- Soft rounded UI
- Doodle aesthetic
- Calm emotional atmosphere

The design should be cute but not childish.

---

## Stitch Design Source

Use the Stitch design as visual inspiration only.

Important:
- Do not copy fake phone mockup frame into production.
- Do not render fake iPhone bezel.
- Do not render fake notch.
- Do not render outer device shadow.
- Use real app layout instead.

The Stitch mockup is a visual reference for:
- colors
- spacing
- typography feel
- mascot direction
- card style
- emotional tone

---

## Layout Direction

Mobile:
- full width
- native mobile app feel
- safe-area aware
- fixed bottom navigation
- no fake phone frame

Desktop:
- black background
- centered mobile-width app container
- max width around 420px
- no device mockup

Recommended container:

```txt
w-full max-w-md mx-auto min-h-screen bg-black
